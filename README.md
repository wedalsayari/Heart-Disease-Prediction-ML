# ❤️ Heart Disease Prediction Using Machine Learning

## 📌 Overview

This project focuses on predicting heart disease using Machine Learning algorithms based on the UCI Heart Disease Dataset. Different classification models were developed, trained, and evaluated to compare their performance and identify the most effective approach for heart disease prediction.

---

## 🎯 Project Objectives

- Predict the presence of heart disease using patient medical data.
- Compare the performance of multiple Machine Learning algorithms.
- Evaluate each model using standard classification metrics.
- Analyze model strengths and limitations.

---

## 📊 Dataset

The project uses the **UCI Heart Disease Dataset**, which includes patient information such as:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Maximum Heart Rate
- Exercise-Induced Angina
- ST Depression
- Number of Major Vessels
- Thalassemia

---

## 🤖 Machine Learning Models

The following algorithms were implemented and compared:

- 🌳 Decision Tree
- 🌲 Random Forest
- 👥 K-Nearest Neighbors (K-NN)

---

## 📈 Model Evaluation

The models were evaluated using:

- Accuracy
- Precision
- Recall
- F1-Score

### Results Summary

| Model | Performance |
|--------|-------------|
| Decision Tree | Highest accuracy on the evaluation dataset |
| Random Forest | Excellent performance with strong generalization |
| K-NN | Lower performance compared to the other models |
